Endpoints

GET/v1/currencies/{base}.json
All exchange rates for the given base currency
GET/v1/currencies/{base}.min.json
Minified version (smaller, faster)
GET/v1/currencies/currencies.json
List of all supported currency codes
GET/v1/currencies/latest.json
All bases × all targets in one file

Quick Start

// JavaScript
const res = await fetch('https://nemesisx1.github.io/currency-api/v1/currencies/usd.json');
const data = await res.json();
const eurRate = data.usd.eur; // 0.85
console.log(100 * eurRate);   // 85 EUR
# Python
import urllib.request, json
url = "https://nemesisx1.github.io/currency-api/v1/currencies/usd.min.json"
data = json.loads(urllib.request.urlopen(url).read())
print(100 * data["usd"]["eur"])  # 85.0
# cURL
curl https://nemesisx1.github.io/currency-api/v1/currencies/usd.json
